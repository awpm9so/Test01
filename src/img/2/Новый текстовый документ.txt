МУЖСКИЕ КРОССОВКИ NIKE REACT SERTU (AT5301-004)

Пол: Мужское

Цвета: Серый

Страна: Вьетнам

Состав: Кожа, синтетика, текстиль, резина

Мужские кроссовки от Nike новой модели React Sertu. Кроссовки, отдающие дань уважения основателю компании Nike Биллу Бауэрману и его военной карьере. Модель выполнена из натуральной замши и кожи с вставками из переплетающихся нейлоновых полосок на амортизирующей подошве из нового материала React. Надписи "The10th" на петле и по бокам кроссовок отсылают к десятой горной дивизии, в которой Билл Бауэрман был командиром.

— натуральная замша и синтетические материалы для верха кроссовок
— пенный материал React для амортизации
— петельчатая прямая шнуровка
— петля на пятке для быстрого надевания
— язычок из прозрачного материала
